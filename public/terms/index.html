Terms

Keystone content is provided for research, education, and evaluation. It should be interpreted as prototype and thesis-context material, not operational certification.

Use carefully

Benchmark results and methodology need context before they influence security decisions.

No certification advice

The site does not provide legal, compliance, or security certification advice. Independent review is required for operational decisions.

Context matters

Benchmark figures depend on algorithm parameters, libraries, hardware, quantum backend behavior, and measurement methodology.

Validation responsibility

Users are responsible for validating results, assumptions, and implementation choices before applying them to production security work.

Product scope

The public site presents Keystone's functional macOS application and verified release status. It does not create accounts, contracts, subscriptions, or managed services.
