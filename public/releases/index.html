Release status

Keystone 1.0.0 for Apple silicon is available for macOS 13.5 or later. The DMG is Developer ID signed, notarized, stapled, and checksum-verified.

Download Keystone 1.0.0

macOS 1.0.0

  • Apple silicon DMG for macOS 13.5 or later.
  • Developer ID signed and accepted by Gatekeeper.
  • Apple notarized with a stapled ticket for offline validation.
  • SHA-256: a79db8d97bbcfcb117f53d65e5df2e3aa4aae3b69123204e578a5cf8a37d8bf8

Future directions

  • Linux timing improvements for broader benchmarking environments.
  • Additional quantum backends and provider adapters.
  • Expanded comparison tooling for algorithm families and parameter sets.
  • More portable reporting around backend noise, circuit depth, and execution context.
